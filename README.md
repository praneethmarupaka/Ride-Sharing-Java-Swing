# 🚖 Ride Sharing App (Java Swing)

A simple desktop-based **Ride Sharing Application** built using **Java Swing**.  
This project demonstrates GUI development in Java along with fare calculation logic and ride management.

---

## 📌 Features

- 🏁 Enter **Current Location** and **Drop Location**
- 🚗 Select **Vehicle Type** (Bike / Car)
- 👥 Choose **Number of Passengers**
- 📏 Input **Distance (in km)**
- 💰 Calculate **Fare dynamically**
- ✅ Confirm and store rides
- 📋 View all confirmed rides

---

## 🛠️ Technologies Used

- **Java**
- **Swing (GUI Framework)**
- **AWT (Event Handling & Layouts)**
